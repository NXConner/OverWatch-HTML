# Prisma Migrations Baseline

This folder stores Prisma migration directories generated by `prisma migrate`.

## Usage

1. Generate Prisma client:
   - `npm run prisma:generate`
2. Create and apply a new local migration:
   - `npm run prisma:migrate:dev -- --name <migration_name>`
3. Apply committed migrations in non-dev environments:
   - `npm run prisma:migrate:deploy`

Keep migration files committed to source control so all environments stay in sync.
